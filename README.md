# Booking System (Go)

## 📌 Project Description
This repository contains a skeleton implementation of a hotel booking platform developed for the **Advanced Programming 1** course.  
The project demonstrates backend architecture in Go, module-based design, and team collaboration using Git.

The system is structured as a **monolithic Go application** and includes modules for authentication, room catalog, and booking logic.

---

## ✨ Planned Features
- User Registration & Login (JWT)
- Rooms & Services Catalog
- Booking Creation & Price Calculation
- Payments & Reviews
- Database Migrations (PostgreSQL)
- Optional HTML Templates for UI

---

## 🛠 Tech Stack
| Component | Technology |
|---|---|
| **Language** | Go 1.21+ |
| **Database** | PostgreSQL |
| **Templating** | Go HTML Templates |
| **Version Control** | Git (multiple branches per member) |
